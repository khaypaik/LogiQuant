# 구글 애드센스 설정 가이드

## 현재 구현 상태

✅ 구글 애드센스 스크립트가 `layout.tsx`에 추가되었습니다.
✅ 광고 컴포넌트(`AdSense.tsx`)가 생성되었습니다.
✅ 페이지에 광고가 배치되었습니다 (결과 아래 + 페이지 하단).

## 다음 단계: 실제 광고 단위 생성 및 설정

### 1. 구글 애드센스에서 광고 단위 생성

1. [구글 애드센스 대시보드](https://www.google.com/adsense/)에 로그인
2. **광고** → **광고 단위** 메뉴로 이동
3. **새 광고 단위** 클릭
4. 다음 설정으로 광고 단위 생성:
   - **이름**: "결과 아래 광고" 또는 "페이지 하단 광고"
   - **광고 형식**: "반응형 표시 광고" (권장)
   - **광고 크기**: "반응형" 선택
5. 생성 후 **광고 슬롯 ID**를 복사합니다 (예: `1234567890`)

### 2. 코드에 광고 슬롯 ID 적용

`src/app/page.tsx` 파일에서 광고 슬롯 ID를 실제 값으로 변경:

```typescript
// 현재 (예시 값)
<AdSense 
  adSlot="1234567890"  // ← 이 값을 실제 광고 슬롯 ID로 변경
  adFormat="auto"
  fullWidthResponsive={true}
  className="rounded-lg"
/>
```

### 3. 광고 배치 위치

현재 다음 위치에 광고가 배치되어 있습니다:

1. **계산 결과 아래** (`page.tsx` 라인 44-50)
   - 계산 결과가 표시된 후에만 광고가 나타남
   - 사용자가 결과를 확인한 시점에 노출

2. **페이지 하단** (`page.tsx` 라인 66-72)
   - 정보 섹션 아래에 배치
   - 페이지 전체를 본 사용자에게 노출

### 4. 광고 개수 조정

필요에 따라 광고 개수를 조정할 수 있습니다:

- **현재**: 2개 (결과 아래 1개 + 페이지 하단 1개)
- **최대 권장**: 3개 (구글 정책상 페이지당 최대 3개 권장)
- **추가 위치 고려**: 사이드바(데스크톱), 중간 콘텐츠(모바일)

### 5. 다크모드 대응

광고 컨테이너는 다크모드에서도 자동으로 스타일이 적용됩니다:
- 라이트 모드: 밝은 회색 배경 (`#f9fafb`)
- 다크 모드: 어두운 회색 배경 (`#1f2937`)

### 6. 성능 최적화

현재 구현:
- `strategy="afterInteractive"`: 페이지 로드 후 광고 스크립트 로드
- 광고는 사용자가 스크롤하여 해당 영역에 도달할 때 로드됨
- 초기 페이지 로딩 속도에 영향 최소화

## 주의사항

⚠️ **광고 슬롯 ID는 반드시 실제 값으로 변경해야 합니다.**
- 현재 `1234567890`은 예시 값입니다.
- 구글 애드센스에서 생성한 실제 광고 슬롯 ID로 변경하지 않으면 광고가 표시되지 않습니다.

⚠️ **광고 정책 준수**
- 클릭 유도 금지
- 광고 클릭을 강요하는 메시지 금지
- 광고와 콘텐츠 구분 명확히

## 테스트 방법

1. 개발 서버 실행: `npm run dev`
2. 브라우저 개발자 도구(F12) → Network 탭 확인
3. `adsbygoogle.js` 파일이 로드되는지 확인
4. Elements 탭에서 `<ins class="adsbygoogle">` 태그 확인
5. 실제 광고가 표시되는지 확인 (광고 슬롯 ID 설정 후)

## 문제 해결

### 광고가 표시되지 않는 경우

1. **광고 슬롯 ID 확인**: 실제 값으로 변경했는지 확인
2. **사이트 승인 상태**: 구글 애드센스에서 사이트 승인이 완료되었는지 확인
3. **광고 차단 프로그램**: 브라우저 확장 프로그램 확인
4. **콘솔 에러 확인**: 개발자 도구 Console 탭에서 에러 메시지 확인

### 광고가 너무 많은 경우

`src/app/page.tsx`에서 불필요한 광고 컴포넌트를 제거하거나 주석 처리하세요.
