# 구글 애드센스 전략 모범 답안

## 1. 광고 배치 전략
**답안**: 계산 결과 표시 영역 아래와 페이지 하단에 광고 배치
- **이유**: 
  - 사용자가 계산 결과를 확인한 후 자연스럽게 광고를 볼 수 있음
  - 입력 폼 영역은 사용자 집중도가 높아 방해하지 않음
  - 결과 확인 후 스크롤 다운 시 추가 정보(광고) 노출

## 2. 광고 유형 선택
**답안**: 반응형 광고(Responsive Ad) 우선 사용
- **이유**:
  - 모바일/데스크톱 자동 최적화
  - 화면 크기에 따라 자동 조정
  - 구글의 AI가 최적 크기 자동 선택

## 3. 광고 개수 제한
**답안**: 페이지당 최대 3개
- **배치 위치**:
  1. 계산 결과 영역 아래 (1개)
  2. 페이지 하단 정보 섹션 아래 (1개)
  3. 사이드바(데스크톱) 또는 중간(모바일) (1개)
- **이유**: 사용자 경험을 해치지 않으면서도 충분한 노출 확보

## 4. 모바일 vs 데스크톱 전략
**답안**: 반응형 광고로 자동 대응, 추가로 데스크톱에는 사이드 광고 고려
- **모바일**: 
  - 결과 아래 수평 배너 (1개)
  - 페이지 하단 수평 배너 (1개)
- **데스크톱**:
  - 결과 아래 수평 배너 (1개)
  - 사이드바 수직 배너 (1개, 선택적)
  - 페이지 하단 수평 배너 (1개)

## 5. 다크모드 대응
**답안**: 다크모드에서도 광고 가시성 유지
- **전략**: 
  - 광고 컨테이너에 적절한 패딩과 배경색 적용
  - 다크모드에서도 광고가 자연스럽게 보이도록 스타일링
  - 구글 애드센스가 자동으로 다크모드 광고 제공

## 6. 로딩 성능 최적화
**답안**: 지연 로드(Lazy Load) 적용
- **전략**:
  - `strategy="lazyOnload"` 사용
  - 사용자가 스크롤하여 광고 영역에 도달할 때 로드
  - 초기 페이지 로딩 속도 유지

## 7. 사용자 행동 기반 배치
**답안**: 계산 결과 표시 후 광고 노출
- **전략**:
  - 계산 결과가 표시된 후에만 광고 로드
  - 사용자가 결과를 확인한 시점에 광고 노출
  - 사용자 참여도가 높을 때 광고 클릭률 증가

## 8. 광고 차단 대응
**답문**: 광고 차단 감지 후 정중한 메시지 표시
- **전략**:
  - 광고 차단 사용자에게 "이 서비스는 무료로 제공되며, 광고 수익으로 운영됩니다" 메시지
  - 강제적이지 않은 톤으로 안내
  - 광고 차단 해제 유도보다는 정보 제공 중심

## 9. A/B 테스트 전략
**답안**: 초기에는 보수적 배치, 이후 데이터 기반 최적화
- **1단계**: 기본 배치로 시작 (결과 아래 + 페이지 하단)
- **2단계**: 2주 후 데이터 분석
- **3단계**: CTR, RPM 데이터 기반으로 위치 조정
- **4단계**: 최적 위치 확정

## 10. 수익 모델 우선순위
**답안**: 사용자 경험 우선, 자연스러운 광고 노출
- **원칙**: 
  - 사용자가 앱을 계속 사용할 수 있도록 UX 우선
  - 광고는 콘텐츠의 자연스러운 연장선으로 배치
  - 장기적 사용자 유지가 최종적으로 더 높은 수익으로 이어짐

## 구현 우선순위
1. **1단계**: 계산 결과 아래 반응형 광고 (1개)
2. **2단계**: 페이지 하단 반응형 광고 (1개)
3. **3단계**: 다크모드 스타일링 최적화
4. **4단계**: 성능 모니터링 및 A/B 테스트
